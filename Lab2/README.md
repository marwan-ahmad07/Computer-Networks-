# Application Layer: Email Client GUI (SMTP/IMAP)

## Overview
This project involves the development of a fully functional graphical Email Client using Python. It demonstrates the practical application of application-layer protocols for electronic mail transmission and retrieval.

## Objectives
- Automate email composition and transmission using the SMTP protocol.
- Retrieve and read emails from an inbox utilizing IMAP/POP3 protocols.
- Develop an interactive graphical user interface (GUI) to handle real-time notifications for sent and received messages.

## Technical Details
- **Language**: Python
- **Protocols**: SMTP, IMAP
- **Interface**: Graphical User Interface with real-time notification components.
