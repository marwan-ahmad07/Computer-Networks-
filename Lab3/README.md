# Network Traffic Analysis and Deep Packet Inspection

## Overview
This laboratory is dedicated to network troubleshooting and deep packet inspection using Wireshark. It focuses on understanding data encapsulation across the OSI model layers.

## Objectives
- Utilize Wireshark to intercept and dissect network packets.
- Trace the transmission of specific data payloads across a network.
- Investigate protocol headers and the encapsulation and decapsulation processes.

## Technical Details
- **Primary Tool**: Wireshark
- **Analysis Scope**: OSI Model Layers, Packet Tracing, Protocol Headers
