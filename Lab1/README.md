# Transport Layer: TCP and UDP Socket Programming

## Overview
This laboratory focuses on the implementation of low-level client-server architectures using Python sockets. It explores the foundational differences between connection-oriented and connectionless protocols.

## Objectives
- Implement a reliable, stream-oriented TCP server and client.
- Implement a datagram-oriented, low-latency UDP server and client.
- Analyze the performance and structural differences between TCP and UDP communications.

## Technical Details
- **Language**: Python
- **Key Modules**: `socket`
- **Execution**: Run the server script first to establish the listening socket, followed by the client script to initiate communication.
